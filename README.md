## Framework

简单版php框架

## Framework特点

> 1.    单入口index.php
> 2.    采用面向对象思想，基于MVC设计思想，使用观察者，注册器，工厂，代理，trait特性等模式开发。
> 3.    使用namespace命名空间。
> 4.    autoLoader自动加载类。
> 5.    debug，Profiler(性能分析器)。
> 6.    提供一些常用类(图片,分页,文件上传,验证码等)。
> 7.    提供grunt常用的插件(文件压缩,合并,观察等)。
> 8.    基于ArrayAccess通过数组下标访问配置文件。
> 9.    obcache缓存。
